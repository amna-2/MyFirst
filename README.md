# AI GPA Helper (Assignment App)

This is a simple, ready-to-use AI-style web app for a class assignment.

## What it does
- Accepts user input: course name, credits, and letter grade.
- Calculates a weighted GPA instantly.
- Generates meaningful AI-style study advice based on GPA and weak courses.

## How to run
1. Download/open this project folder.
2. Open `index.html` directly in your browser.

No installation required.
